# ADR 010 — No Anthropic Trademark Public

## Synthesis

ADR 010 = decision Andura WILL NOT reference Anthropic / Claude / Anthropic trademark public-facing material. Original LOCK V1 2026-04-25 (Status: Active). Supersedes implicit branding 24 Apr 2026 "Claude AI Opus 4.7 Coach" terminology cleanup. Trigger: legal verification Anthropic Consumer Terms of Service + Trademark Policy revealed naming pattern violates policy (requires explicit written permission marketing@anthropic.com, NOT obtained pentru bootstrap phase Andura). Public brand = **Andura**. Public tech positioning = generic AI terminology ("AI coaching engine"). Internal docs (vault, ADRs, technical specs) factual references OK + Privacy Policy / TOS factual disclosure GDPR + editorial third-party OK + code comments model identifiers OK. NOT allowed: brand names "Claude/Anthropic", logos, color palettes matching Anthropic brand, "Powered by Claude" taglines, SEO keyword promotion. Strategic benefits: **vendor independence** (Mythos future / cost re-platform NO brand break) + **differentiation** (outcomes sell, NU vendor) + **risk mitigation** (clean trademark posture pre-acquisition due diligence) + **industry analogy** (Coca-Cola Brazilian sugar NU advertised, Stripe AWS backbone NU advertised). Reconsideration triggers: explicit written permission Anthropic, official partner program brand usage rights, "Built on Claude" official badge program, legal counsel nominative fair use determination.
