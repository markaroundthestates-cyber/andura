# Andura — Exercise demo image prompts (2×2 sheets, copy-paste ready)

**Scop:** o imagine = un grid 2×2 cu 4 exerciții (4 panouri). Acoperă toate cele **143 CORE_AUTO** (= exact setul pe care engine-ul îl recomandă automat după gate-ul CORE). 39 de imagini.

## Reguli de consistență (CITEȘTE întâi)
- **Folosește același SEED** la fiecare sheet (sau referința de personaj a tool-ului: Midjourney `--cref`/`--seed`, DALL·E "same character", etc.). Blocul de STIL e identic verbatim în fiecare prompt — asta + seed-ul fix dau **același manechin + același fundal** peste tot.
- **Sunt FOTO, nu video** — pose statice la contracție. Fără timing/animație.
- **Tempo / pauză / variații de tehnică = ACELAȘI exercițiu** → NU au poză separată (sunt MODIFIER, nici nu intră în cele 143). „Flat Bench" și „Flat Bench tempo" = aceeași poză.
- După generare, **taie fiecare sheet în 4 pătrate** = 4 poze individuale, una per exercițiu.
- Slotul de poză din app e mai mic decât ecranul → un pătrat curat (~1:1) e perfect.

## STYLE (e deja inclus în fiecare prompt mai jos — nu trebuie să-l lipești separat)
> Photorealistic gym exercise demo sheet, 2×2 grid of four equal square panels split by thin white lines. EVERY panel uses the SAME single androgynous matte light-grey 3D mannequin (featureless face, athletic build), the SAME seamless dark-charcoal studio background, the SAME soft frontal studio lighting, the SAME 3/4 front camera, full body in frame, identical scale. Each panel = ONE exercise frozen at peak-contraction mid-rep, correct equipment, clean form. No text, no numbers, no logos. Use the SAME seed across all sheets.
