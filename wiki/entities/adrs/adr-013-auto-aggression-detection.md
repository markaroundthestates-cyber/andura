# ADR 013 — Auto-Aggression Detection

## Synthesis

ADR 013 = decision auto-aggression detection sistem pattern recognition self-sabotage (volume creep neprovocat / calorie restriction acceleration / frustration markers / ignore recovery signals / recovery debt cronic) + Hyperfocus amplificator per-profil + 4 profiles (Sprinter / Marathon / Yo-yo / Strategic) + reconciliation hybrid onboarding (self-report 3-4 întrebări + 1 behavioral post-prima sesiune + 4-6 săpt observation + reconciliation prompt). Original LOCK V1 2026-04-26. 5 signals comportamentale + 1 amplificator hyperfocus (8h+ app/zi 4+ zile/săpt) calibrare thresholds stricter per-profil. Composite fatigue score 3 signals (reps<60% 2+ exerciții, rating ≤2/5 proxy temporar, volume scădere voluntară >20% fără reason CDL). 3 severity tiers: **LOW** (1 signal isolat — log CDL no notify) → **MED** (2-3 signals 2 săpt — soft warning banner dismissible) → **HIGH** (3+ signals 1 săpt + escalation pattern — Intervention C). Intervention model: **Soft B default** (observation → pattern → return decision to user, wording template "metric X arată Y. Pattern: Z. Decizia ta?") + dismiss memory anti alarm fatigue. **§AMENDED 2026-04-30** Intervention C hard intervention force-typing string ELIMINATED PERMANENT — replaced cu modal blocking (no scroll-skip) + wording neutral evidence-based + dismiss memory persistence + Liability Flag silent backend CDL `outcome.autoAggression.escalating: true` + `override_acknowledged: timestamp` (NU afișat user — escalation detection viitoare + audit trail liability legal + calibrare empirică thresholds post 50+ users). Rationale eliminare: **Anti-paternalism ABSOLUTE** + **Anti-RE protection categorical** (force-typing string expose internals tech-lifters target RE) + UX hostility prevention + HIGH tier intervention strength preserved alt path.
